# הורדת הקבצים המעודכנים לבדיקה

הקבצים המעודכנים נמצאים בבקשת המשיכה של הענף הנוכחי. כדי לבדוק אותם במחשב שלך אפשר להוריד את כל הפרויקט כ-ZIP או להוריד רק את קובצי הקוד הדרושים.

## הקבצים הדרושים להרצה

כדי להריץ את התוכנה המעודכנת במחשב, ודא שיש לך באותה תיקייה לפחות את הקבצים הבאים:

- `gui.py`
- `file_renamer.py`
- `requirements.txt`
- `run.bat` או `run.sh` — אופציונלי, רק לקיצור הרצה

קבצים נוספים שעודכנו לתיעוד/בדיקה:

- `README.md`
- `INSTRUCTIONS.md`
- `examples.py`
- `tests/test_file_renamer.py`

## דרך 1: הורדת כל בקשת המשיכה כ-ZIP

1. פתח את בקשת המשיכה ב-GitHub.
2. לחץ על הכפתור הירוק **Code**.
3. בחר **Download ZIP**.
4. חלץ את קובץ ה-ZIP במחשב.
5. פתח Command Prompt/PowerShell בתיקייה שחולצה.
6. הרץ:

```bat
python gui.py
```

## דרך 2: הורדת קבצים בודדים מ-GitHub

1. פתח את בקשת המשיכה.
2. עבור ללשונית **Files changed**.
3. פתח כל אחד מהקבצים הדרושים.
4. לחץ **Raw**.
5. שמור את הקובץ באותה תיקייה במחשב שלך.
6. בסיום, הרץ:

```bat
python gui.py
```

## דרך 3: הורדה עם Git

אם מותקן אצלך Git, אפשר להריץ:

```bat
git clone <כתובת-המאגר>
cd Hebrew-numbering
git checkout <שם-הענף-של-בקשת-המשיכה>
python gui.py
```

## בדיקה מהירה של הפיצ'רים החדשים

בממשק הגרפי:

- למספור עברי מעל אלף בחר **עברית**, ואז בחר סגנון אלפים: `אותיות (א' א')` או `מילים (אלף א')`.
- למספור עם אפסים מובילים בחר מספור מספרי, ואז בחר `0 אחד (01, 02)` או `00 שניים (001, 002)`.
