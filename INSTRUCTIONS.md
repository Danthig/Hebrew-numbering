תוכנת שינוי שם קבצים - הוראות שימוש
========================================

## מהי התוכנה?

תוכנה זו מאפשרת לשנות שם של קבצים מרובים בבת-אחת בדרכים שונות:
- מספור בעברית (א' ב' ג' וכו')
- מספור רגיל (1 2 3 וכו')
- הוספת התחילית משותפת
- בחירה בתו הפרדה

## דרישות מוקדמות:
- Python 3.7 ומעלה
- tkinter (בדרך כלל כלול ב-Python)

## התקנה ראשונה:

### Windows:
1. הורד Python מ- https://www.python.org
   * בחשוב: בחר "Add Python to PATH" במהלך ההתקנה
2. פתח Command Prompt או PowerShell בתיקייה של התוכנה
3. הרץ: `python gui.py` או לחץ על `run.bat`

### Linux/macOS:
1. בדוק אם Python 3 מותקן: `python3 --version`
2. בדוק אם tkinter מותקן:
   - Ubuntu/Debian: `sudo apt-get install python3-tk`
   - macOS: `brew install python-tk`
3. הרץ: `python3 gui.py` או `bash run.sh`

## שימוש בממשק הגרפי:

### שלב 1: בחירת תיקייה
- לחץ על "בחר תיקייה"
- בחר את התיקייה שמכילה את הקבצים שברצונך להינם

### שלב 2: בחירת סוג המספור

**עברית עם גרש:**
- א' ב' ג' ד' ה' ... ט"ו ט"ז י"ז ... צ"ט

**עברית ללא גרש:**
- א ב ג ד ה ... טו טז יז ... צט

**מספרים עם נקודה:**
- 1. 2. 3. 4. 5. ... 10. 11. ...

**מספרים ללא נקודה:**
- 1 2 3 4 5 ... 10 11 ...

### שלב 3: בחירת התחילית וההפרדה

**התחילית:** 
- זה הטקסט שיהיה בתחילת כל קובץ
- דוגמה: "קובץ", "תמונה", "מסמך"

**תו הפרדה:**
- זה מפריד בין ההתחילית למספור
- אפשרויות נפוצות: _, -, (רווח)
- דוגמה: קובץ_א' או קובץ-א' או קובץ א'

### שלב 4: התחלת המספור
- אתה יכול להתחיל מהמספר שרוצה
- דוגמה: אם בחרת 10, הקובץ הראשון יהיה קובץ_י'

### שלב 5: תצוגה מקדימה
- לפני ביצוע, תראה בדיוק איך יראו השמות החדשים
- אם משהו לא נכון, שנה את ההגדרות

### שלב 6: ביצוע השינוי
- לחץ על "בצע שינוי שם!"
- בחר "כן" כדי לאשר
- ההתוכנה תשנה את שם כל הקבצים

## דוגמאות שימוש:

### דוגמה 1: תמונות מחופשה

**הגדרות:**
- סוג מספור: עברית עם גרש
- התחילית: "תמונה"
- תו הפרדה: _
- התחלה: 1

**תוצאה:**
- photo1.jpg → תמונה_א'.jpg
- photo2.jpg → תמונה_ב'.jpg
- photo3.jpg → תמונה_ג'.jpg

### דוגמה 2: מסמכים ממוספרים

**הגדרות:**
- סוג מספור: מספרים עם נקודה
- התחילית: "מסמך"
- תו הפרדה: (רווח)
- התחלה: 1

**תוצאה:**
- doc1.docx → מסמך 1.docx
- doc2.docx → מסמך 2.docx
- doc3.docx → מסמך 3.docx

### דוגמה 3: ווידאו מספור אחורי

**הגדרות:**
- סוג מספור: מספרים ללא נקודה
- התחילית: "clip"
- תו הפרדה: -
- התחלה: 10

**תוצאה:**
- video1.mp4 → clip-10.mp4
- video2.mp4 → clip-11.mp4
- video3.mp4 → clip-12.mp4

## טיפים וטריקים:

### טיפ 1: התחילית ריקה
אם תרוך את שדה התחילית ריק, רק המספור יתווסף:
- קובץ1.txt → א'.txt
- קובץ2.txt → ב'.txt

### טיפ 2: תו הפרדה שונה
אתה יכול להשתמש בכל תו שרוצה כתו הפרדה:
- קובץ_א'
- קובץ-א'
- קובץ.א'
- קובץ:א'
- קובץ|א'

### טיפ 3: המספור בעברית
המספור בעברית תומך עד 9999:
- 1-9: א' עד ט'
- 10-19: י' עד י"ט
- 20-90: כ' עד צ'
- 100-900: ק' עד תת"ק
- 1000-9999: אלף עד תתקצ"ט

### טיפ 4: בדיקה לפני ביצוע
תמיד בדוק את תצוגה המקדימה לפני לחיצה על "בצע שינוי שם!"
- ודא שהשמות החדשים הם כמו שרוצים
- בדוק שאין קבצים שכבר יש להם את השם החדש

### טיפ 5: גיבוי מומלץ
לפני שינוי שם של קבצים חשובים:
1. צור גיבוי של התיקייה
2. הרץ את התוכנה
3. בדוק את התוצאות

## פתרון בעיות:

### בעיה: "התיקייה לא נמצאה"
- בדוק שהתיקייה קיימת
- וודא שלא מחקת אותה
- נסה לבחור שוב

### בעיה: "שגיאה בשינוי השם"
- בדוק שיש לך הרשאות כתיבה בתיקייה
- ודא שאף קובץ לא פתוח בתוכנה אחרת
- נסה להריץ את התוכנה כמנהל (Windows)

### בעיה: "קובץ עם אותו שם כבר קיים"
- בדוק שלא יש קובץ שכבר יש לו את השם החדש
- אם צריך, הוסף מספור שונה או התחילית שונה

## שימוש מתקדם - Python API:

אם אתה מתכנת, אתה יכול להשתמש בקוד ישירות:

```python
from file_renamer import FileRenamer, HebrewNumbering

# יצירת renamer
renamer = FileRenamer("C:\\my\\folder")

# קבלת קבצים
files = renamer.get_files()

# שינוי שם
renamer.rename_files(
    files,
    prefix="קובץ",
    numbering_type="hebrew",  # או "numeric" או "numeric_no_dot"
    with_geresh=True,
    separator="_",
    start_number=1
)

# המרה של מספר לעברית
number = HebrewNumbering.number_to_hebrew(5, with_geresh=True)
print(number)  # ה'
```
