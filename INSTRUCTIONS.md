תוכנת שינוי שם קבצים - הוראות שימוש
========================================

## מהי התוכנה?

התוכנה מאפשרת לשנות שם של קבצים מרובים בבת אחת:

- מספור בעברית, כולל מעל 1,000.
- **מספור בלשון נקייה** (שד→דש, ובאופציה רע→ער, שמד→דמש).
- מספור רגיל (`1`, `2`, `3`) ועם אפסים מובילים (`01`, `001`).
- הוספת תחילית משותפת ובחירת תו הפרדה.
- החלפת השם המקורי או הוספת המספור לפני/אחרי השם.
- **ביטול (Undo)** לריצה האחרונה — גם לאחר סגירת התוכנה.
- **שינוי בטוח (אטומי)** עם זיהוי התנגשויות.
- **ממשק גרפי מודרני** (HTML/CSS) עם מצב כהה, או ממשק tkinter כגיבוי.
- **שורת פקודה (CLI)** מלאה.

## איפה קובץ ה-EXE?

אם קובץ ההפעלה כבר נבנה, הוא נמצא בתיקיית הפרויקט הראשית בשם:

```text
שינוי_שם_קבצים.exe
```

### בניית קובץ ה-EXE מחדש

```bat
build_exe.bat
```

או ידנית:

```bat
pip install pyinstaller pywebview
pyinstaller hebrew_renamer.spec
```

הקובץ ייווצר בתיקיית `dist\`.

### הרצה ישירות מהקוד (ללא EXE)

```bat
python gui.py
```

## דרישות מוקדמות

- Python 3.9 ומעלה.
- לממשק המודרני: `pip install pywebview` (אופציונלי — יש נפילה אוטומטית ל-tkinter).
- ל-tkinter (גיבוי): בדרך כלל כלול בהתקנת Python.

## התקנה והרצה

### Windows

1. התקן Python מאתר https://www.python.org.
2. במהלך ההתקנה סמן `Add Python to PATH`.
3. פתח Command Prompt או PowerShell בתיקיית התוכנה.
4. הרץ:

```bat
python gui.py
```

או:

```bat
run.bat
```

### Linux/macOS

1. בדוק ש-Python מותקן:

```bash
python3 --version
```

2. אם tkinter לא מותקן:
   - Ubuntu/Debian: `sudo apt-get install python3-tk`
   - macOS: `brew install python-tk`
3. הרץ:

```bash
python3 gui.py
```

או:

```bash
bash run.sh
```

## שימוש בממשק הגרפי

1. לחץ על **בחר תיקייה** או **בחר קבצים**.
2. בחר את סוג המספור:
   - **עברית**
   - **1.**
   - **1**
3. למספור עברי מעל אלף, בחר:
   - **אותיות (א' א')**
   - **מילים (אלף א')**
4. למספור מספרי עם אפסים מובילים, בחר:
   - **ללא (1, 2, 3)**
   - **0 אחד (01, 02)**
   - **00 שניים (001, 002)**
5. בחר תחילית, תו הפרדה ומספר התחלה.
6. בדוק את התצוגה המקדימה.
7. לחץ על **בצע שינוי שם!**.

## דוגמאות

- עברית מעל אלף בסגנון אותיות: `1001` → `א' א'`
- עברית מעל אלף בסגנון מילים: `1001` → `אלף א'`
- אפס אחד: `1, 2, 3, 10, 11` → `01, 02, 03, 10, 11`
- שני אפסים: `1, 2, 3, 10, 11, 100` → `001, 002, 003, 010, 011, 100`

## מספור בלשון נקייה

הפעל את המתג **"מספור בלשון נקייה"** כדי להחליף צירופי אותיות לא רצויים:

- `304` → `ד"ש` (במקום `ש"ד`)
- באופציה מורחבת: `270` → `ע"ר`, `344` → `דמ"ש`

## שורת פקודה (CLI)

```bat
REM תצוגה מקדימה בלבד
python -m hebrew_renamer ./photos --prefix תמונה --type hebrew --dry-run

REM מספור עברי בלשון נקייה
python -m hebrew_renamer ./photos --clean-language

REM ביטול הריצה האחרונה
python -m hebrew_renamer --undo
```

## ביטול (Undo)

לאחר שינוי שם אפשר ללחוץ על **"בטל שינוי אחרון"** כדי להחזיר את כל
הקבצים לשמם המקורי — גם לאחר סגירת התוכנה ופתיחתה מחדש.

## חשוב

- השינוי בטוח ומתועד — תמיד אפשר לבטל עם **Undo**.
- קובץ קיים לא יידרס: שם יעד תפוס מסומן כהתנגשות (באדום) ולא מבוצע.
- בכל זאת, לפני שינוי שמות מרובים, מומלץ לגבות קבצים חשובים.
