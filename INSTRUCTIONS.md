תוכנת שינוי שם קבצים - הוראות שימוש
========================================

## מהי התוכנה?

התוכנה מאפשרת לשנות שם של קבצים מרובים בבת אחת:

- מספור בעברית, כולל מעל 1,000.
- מספור רגיל (`1`, `2`, `3`).
- מספור עם אפסים מובילים (`01`, `02`, `001`, `002`).
- הוספת תחילית משותפת.
- בחירת תו הפרדה.
- החלפת השם המקורי או הוספת המספור לפני/אחרי השם המקורי.

## איפה קובץ ה-EXE?

אם קובץ ההפעלה קיים, הוא נמצא בתיקיית הפרויקט הראשית בשם:

```text
שינוי_שם_קבצים.exe
```

אם יש לך רק את קובצי הקוד, אפשר להריץ כך:

```bat
python gui.py
```

## דרישות מוקדמות

- Python 3.7 ומעלה.
- tkinter, שבדרך כלל כלול בהתקנת Python.

## התקנה והרצה

### Windows

1. התקן Python מאתר https://www.python.org.
2. במהלך ההתקנה סמן `Add Python to PATH`.
3. פתח Command Prompt או PowerShell בתיקיית התוכנה.
4. הרץ:

```bat
python gui.py
```

או:

```bat
run.bat
```

### Linux/macOS

1. בדוק ש-Python מותקן:

```bash
python3 --version
```

2. אם tkinter לא מותקן:
   - Ubuntu/Debian: `sudo apt-get install python3-tk`
   - macOS: `brew install python-tk`
3. הרץ:

```bash
python3 gui.py
```

או:

```bash
bash run.sh
```

## שימוש בממשק הגרפי

1. לחץ על **בחר תיקייה** או **בחר קבצים**.
2. בחר את סוג המספור:
   - **עברית**
   - **1.**
   - **1**
3. למספור עברי מעל אלף, בחר:
   - **אותיות (א' א')**
   - **מילים (אלף א')**
4. למספור מספרי עם אפסים מובילים, בחר:
   - **ללא (1, 2, 3)**
   - **0 אחד (01, 02)**
   - **00 שניים (001, 002)**
5. בחר תחילית, תו הפרדה ומספר התחלה.
6. בדוק את התצוגה המקדימה.
7. לחץ על **בצע שינוי שם!**.

## דוגמאות

- עברית מעל אלף בסגנון אותיות: `1001` → `א' א'`
- עברית מעל אלף בסגנון מילים: `1001` → `אלף א'`
- אפס אחד: `1, 2, 3, 10, 11` → `01, 02, 03, 10, 11`
- שני אפסים: `1, 2, 3, 10, 11, 100` → `001, 002, 003, 010, 011, 100`

## חשוב

לפני שינוי שמות מרובים, מומלץ לגבות את הקבצים.
